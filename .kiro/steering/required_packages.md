# Required packages

* **selenium** - use this to scrape data from websites.
* **pandas** - when the ratios are retieved they should be initially added to a Pandas Dataframe before being rendered to the output.
* **tabulate** - when rendering the table output use this for smart column alignment.
* **rich** - use this to add color and style to the output.
* **beautifulsoup4** - to pull data out of HTML and XML files if needed.
